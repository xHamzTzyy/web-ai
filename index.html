Install YAREUU AI CLI
Jalankan YAREUU AI langsung dari terminal kamu. Butuh Node.js v14+. Simpan file yareuu-cli.js lalu jalankan perintah di bawah.
1
Download file CLI
2
Pastikan Node.js terinstall
node --version
3
Jalankan CLI
node yareuu-cli.js
4
(Opsional) Jadikan executable global
chmod +x yareuu-cli.js
./yareuu-cli.js
